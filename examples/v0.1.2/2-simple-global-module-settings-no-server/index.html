Global Module Settings (no server)


This example demonstrates global module settings, which are set using angular values.

Here we have created a User model using the AngularCakePHP data model (scripts/user-model.js),

and applied the following custom settings:

key value/result notes
AngularCakePHPApiUrl {{api}}  
AngularCakePHPApiEndpointTransformer {{api_endpoint}}  
AngularCakePHPUrlParameterTransformer you will see a failed a get request with the transformed url params
